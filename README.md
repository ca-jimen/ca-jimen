# Carlos Jimenez
**`Software Engineer · San Diego, CA`**

I build tools that work in the real world — from **multi-platform desktop apps for the San Diego Fire Department** to personal automation systems and trading tools. I care about reliability, clean architecture, and shipping things that actually get used.

Currently working with **Flutter · Python · C# · RabbitMQ** on mission-critical public safety software, and building side projects ranging from a **Discord-based stock scanner** to a self-hosted portfolio on a Raspberry Pi.

- 🚒 Building CADSOS enhancements and incident map layers for the fire department
- 📈 Developing a real-time stock scanner with Polygon.io + Discord alerts
- 🌐 Launching a personal portfolio with React, TypeScript, and Vite
- 🔒 Self-hosted infrastructure enthusiast (WireGuard, Raspberry Pi, no-cloud-preferred)
